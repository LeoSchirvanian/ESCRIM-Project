Pour pouvoir lancer le code java et se connecter à la base de données, il faudra créer une
nouvelle base de données à l'aide de Wampserver, et entrer les requêtes SQL, ci joint dans
le dossier, afin de créer la même base de données que nous sur votre ordinateur.

Pour se connecter, rentrer comme id : "u" et comme mot de passe : "p" (ou créer votre propre compte
dans votre base SQL).

Le code java requiert aussi d'avoir le package Swing pour être lancer (interface graphique).
